Псевдо селектори

  • :active, :hover, :focus, :visited, :link
  • :nth-child(n), :first-child, :last-child
  • :first-of-type, :last-of-type
  • :only-child (елементът, който е единствено дете на своя родител)
  • :only-of-type (елемент, който няма братовчеди(siblings) от своя тип)
  • :empty (елементът няма деца)
  • ::after (вмъква съдържание след елемента)
  • ::after (вмъква съдържание преди елемента)
  • ::first-letter (първа буква от елемента)
  • ::first-line (първи ред от елемента)
  • ::selection (маркираното от потребителя съдържание)

Opacity

  • Зaдава прозрачност на елемента.
  • Число в интервала [0.0,1.0].
img {
  opacity: 0.4;
  filter: alpha(opacity=40); /* For IE8 and earlier */
}

Visibility

Възможните стойности са visible/hidden.

Разликата между използването на visibility: hidden и display: none e, че за елемент с display: none не се заделя място в страницата, докато за елемнт с visibility: hidden се заделя място.

Още правила за текст

Със CSS е възможно също да конкролираме дали текста да е с главни, малки букви, или всяка дума да започва с главна буква:

p.uppercase {
  text-transform: uppercase;
}
p.lowercase {
  text-transform: lowercase;
}
p.capitalize {
  text-transform: capitalize;
}

CSS Sprite

  • С цел оптимизация при писането на CSS се прилагат някои практики. Една от най важните е използването на CSS Sprite
  • Той намялява броя на request-ите за картинки
  • Много картинки се сливат в една. После чрез background-position се достъпва оригиналната картинка.

CSS3 Gradients

  • background: -moz-linear-gradient(black, white); /* FF 3.6+ */
  • background: -webkit-linear-gradient(black, white); /* Safari 5.1+, Chrome 10+ */
  • background: -o-linear-gradient(black, white); /* Opera 11.10 */
  • filter: progid:DXImageTransform.Microsoft.gradient(startColorstr='#000000', endColorstr='#ffffff'); /* IE6 - IE9 */
  • background: linear-gradient(black, white); /* the standard */

Box Sizing

Възможните стойности са content-box/border-box

  • content-box – по подразбиране. Широчината и височината на елемента не включват неговия padding, border или margin
  • border-box – широчината и височината на елемента включват padding и border /default box model на стари версии на IE/

Responsive design

Риспонсив дизайна прави страницата да изглежда добре на всички устройства – десктоп, таблет, телефон.

Риспонсив дизайна вкючва промяна на големината на елементи, скриване на елементи, намаляване, увеличаване или преместване на съдържание с цел страницата да изглежда добре и да е удобна за използване на различните устройства.

CSS3 Media Queries

Чрез Media Queries можем да задаваме правила, които да се изпълняват на различни условия

@media screen and (min-width: 480px) {
  body {
    background-color: lightgreen;
  }
}

CSS3 Media Queries

@media not|only mediatype and (media feature) {
    CSS-Code;
}

Mediatypes

  • all
  • screen – за екрани на компютри, таблети, смартфони и др
  • print – за принтиране

Media Features

  • width – широчината на viewport- a
  • height – височината на viewport- a
  • max-width
  • max-height
  • min-width
  • min-height

CSS3 Media Queries

Използване на @media за риспонсив дизайн:

@media only screen and (max-width: 500px) {
  .gridmenu { width:100%; }
}
