Основи на HTTP

или как работи уеб-а

  • Hypertext Transfer Protocol
  • Stateless (не пази състояние)
  • Комуникация между разпределени системи (distributed systems)
  • Основите на съвременния уеб
  • Ще го прегледаме отгоре-отгоре

HTTP

Request/Response

HTTP

URLs (част 1)

  • Uniform Resource Locators
  • Връзка към уеб ресурс
  • Идентификатор
  • Проста структура

HTTP

URLs (част 2)

  • Протоколът може да бъде http, https, ftp и др.
  • Портът по подразбиране е 80
  • Resource path - локален път към ресурса на сървъра

HTTP

Verbs/Methods

Действия, които да бъдат осъществени върху сървъра

  • GET - Вземи съществуващ ресурс
  • POST - Създай нов ресурс
  • PUT - Обнови съществуващ ресурс
  • DELETE - Изтрий съществуващ ресурс
  • Има и други (например HEAD, TRACE, OPTIONS)
  • URL-ът съдържа необходимата за сървъра информация, относно желания ресурс

Основи на HTML

или как да структурираме уеб-а

  • HyperText Markup Language
  • Implemented by the browser
  • Основата на всяка уеб страница
  • Задава структура

Пример за HTML страница

Запишете го във файл index.html и го отворете в браузъра.

<!DOCTYPE html>
  <html>
  <head>
    <title>My title</title>
  </head>
  <body>
    <h1>My header</h1>
    <a href="http://example.com">My Link</a>
  </body>
</html>

HTML

Изграждащи единици

Тагове, атрибути, елементи.

<tag> - отварящ таг
</tag> - затварящ таг
<tag attribute="value"> - атрибут на таг
<tag attribute="value">tag content</tag> - елемент

Атрибутите дават допълнителна информация за HTML елементите.

HTML

Структура

<!DOCTYPE html> - Декларира HTML версията на документа (в случая HTML 5)
<head> - Съдържа заглавие, скриптове, стилове, мета информация и други.
<title> - Заглавие на страницата. Съдържа се в <head>.
<body> - Дефинира съдържанието на документа.

Заглавия

<h1>This is heading 1</h1>
<h2>This is heading 2</h2>
<h3>This is heading 3</h3>
<h4>This is heading 4</h4>
<h5>This is heading 5</h5>
<h6>This is heading 6</h6>

Параграф

<p>Paragraph with text.</p>

Номериран списък

<ol>
    <li>First list item</li>
    <li>Second list item</li>
    <li>Third list item</li>
</ol>

Неномериран Списък

<ul>
    <li>One list item</li>
    <li>Some list item</li>
    <li>Another list item</li>
</ul>

Хиперлинк

<a href="http://example.com" title="Link to example.com">My Link</a>

Изображения

<img src="http://example.com/image.jpg" alt="Example Image" />

Секции в страницата

<span>Some text here.</span>

Секции в страницата

<div>Some text here.</div>
